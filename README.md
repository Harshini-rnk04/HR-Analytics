
# HR Analytics Dashboard – Power BI Project

## Project Overview
This project analyzes HR data to help organizations understand employee trends,
headcount distribution, and attrition using Power BI.

## Dashboard Components
- Summary Page with key HR metrics
- Year-on-Year Headcount and Attrition trends
- Headcount analysis by Business Unit, Department, Job Title, Gender, and Age Group
- Interactive slicers for Department, Country, and Employee Name

## Tools Used
- Power BI
- Power Query
- DAX
- Python (for documentation automation)

## Outcome
The dashboard enables data-driven HR decisions and workforce planning.
